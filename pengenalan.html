Apa itu HTML

Ini fondasi dasar sebuah website. Ibarat bangun rumah: HTML itu rangkanya. Dia ngatur “isi”-nya apa aja—judul, paragraf, gambar, tombol, form, dan seterusnya. Tanpa HTML, nggak ada struktur.


Apa itu CSS

Kalau HTML itu rangka, CSS itu “tukang dekorasinya”. Dia ngatur tampilan: warna, ukuran tulisan, posisi elemen, jarak-jarak, layout, sampai bikin website jadi enak dipandang. HTML bikin rumah berdiri, CSS bikin rumahnya cakep.

Apa itu Javascript

Ini bagian yang bikin website “hidup”. Kayak listriknya. Dengan JS, tombol bisa berfungsi, ada animasi, form bisa dicek otomatis, menu bisa sliding, dan halaman bisa berubah tanpa reload. Intinya: bikin website jadi interaktif.

jadi singkatnya:
  • HTML = KERANGKA
  • CSS = TAMPILAN
  • JAVASCRIPT = INTERAKSI
